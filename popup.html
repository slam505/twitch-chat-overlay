Setup Instructions
1. Enable WebSocket in OBS: Tools → WebSocket Server Settings
2. Add URL: https://slam505.github.io/twitch-chat-overlay/
3. Add a Browser Source named TwitchHighlight
4. Set size to 700 × 200
5. Click Save & Connect above
6. Go to Twitch and click ✨ on any chat message!
